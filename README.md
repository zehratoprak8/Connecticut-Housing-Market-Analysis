# Connecticut Housing Market Analysis
A data science project analyzing 43,470 real estate records from Connecticut in 2022 to uncover housing price trends and predict home sale prices using machine learning.
Tools & Technologies

Python, Pandas, Scikit-learn, Matplotlib

# What This Project Does

Cleaned and prepared a large government dataset handling missing values and outliers
Built and compared 3 machine learning models to predict home sale prices
Decision Tree achieved the best accuracy with a Test R² of 0.73

# Files

final_project.py — main Python code
ct_housing_2022.csv — original dataset
ct_housing_2022_clean.csv — cleaned dataset
Connecticut_Housing_Sales_Analysis.pdf — full project report

# Results
The Decision Tree model successfully predicted a sample home in Hartford at ~$294K which was very realistic for that area.
